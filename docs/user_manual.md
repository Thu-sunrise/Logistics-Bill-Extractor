# LOGISTICS BILL EXTRACTOR USER MANUAL

The **Logistics Bill Extractor** is a software designed to automate the extraction of data from PDF Bill of Lading (B/L) files across multiple shipping lines (ONE, HMM, SJJ, MSC, OOCL, ZIM, etc.) and output a synthesized Excel file quickly and accurately.

---

## I. INSTALLATION GUIDE

1. Download the installation file **`Setup_LogisticsBillExtractor_v1.0.0.exe`** to your computer.
2. Double-click the file to begin installation.
3. Click **Next** until the installation is complete. The software icon will appear on your Desktop.

> **WINDOWS SMARTSCREEN WARNING:**
> The first time you open the installer or software, Windows may display a blue warning screen stating *"Windows protected your PC"* (or a false positive from antivirus software). **This is normal** as the internal software has not purchased an expensive digital certificate from Microsoft.
> 
> **How to resolve:**
> - On the blue screen, click the **"More info"** text on the left side.
> - Then, a **"Run anyway"** button will appear on the bottom right. Click it to run the software normally. The system will remember this choice and will not ask again.

---

## II. HOW TO USE

1. Open the software by double-clicking the `LogisticsBillExtractor` icon on your Desktop.
2. At the main interface, click the **"Select Folder"** button.
3. Choose the folder containing the PDF Bill of Lading files you need to extract. (For example: "July Bills").
4. The software will immediately scan the folder and begin batch processing. A progress bar will track the progress.
5. Upon reporting **Complete**, the software will automatically generate an Excel file named `Logistics_Extracted_[Timestamp].xlsx` in the same directory you selected.
6. Open the Excel file. All data (Consignee, Notify party, Vessel name, POL, POD, Description, Remark, etc.) will be neatly organized into columns!

---

## III. AUTO-UPDATE FEATURE

The software has a built-in Auto-Update feature. 
Every time you open the software, it silently checks the system. If the development team releases a new version, the software will display a prompt: *"A new version is available... Do you want to update?"*.
Simply click **"Yes"**, and the software will automatically download the patch, install it, and restart the latest version for you in less than 10 seconds!
